# TSP---berlin52
TSP problem using berlin 52 dataset

Tweaking between:
1. Selection (Tournament, Roulette)
2. Crossover (One-Point, Two-Point)
3. Mutation (Swap, Bit-Flip)

Can also change variables (pop_size, generation, mutation rate) thanks to a slider

How to run?
pip install numpy pandas gradio matplotlib 

Than run the main.py file
